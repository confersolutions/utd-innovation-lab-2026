## 👤 INDIVIDUAL TASKS

### Rujul Shukla — JSON Conversation Trees
Build complete JSON conversation trees for three user flows:
1. **Visitor Inquiries** — General questions about temple, events, timings
2. **Temple Directions** — How to get to the temple, parking, public transit
3. **Donation Requests** — Making donations, tax receipts, recurring giving

**Deliverable:** Three JSON files mapping every decision point, user input type, and bot response path.

---

### Chanakya Alluri — Voice-to-WhatsApp Scripts
Write exact scripts for:
1. **Voice-to-WhatsApp handoff flows** — How users transition from voice call to chat
2. **Fallback messages** for five error scenarios:
   - Intent not understood
   - API timeout
   - Confusing user input
   - Context lost
   - Failed escalation to human

**Deliverable:** Markdown document with all scripts and handoff logic.

### Ananth Vangala — State Machine Flowcharts

3. **Error recovery paths** — How the bot handles failures gracefully

**Deliverable:** Flowchart diagrams (ASCII art, Mermaid, or linked images) with explanations.

### Nikita Pal — Localization Decision Trees

**Deliverable:** Decision trees showing language handling logic.
