## 📋 Weekly Submission Process

1. Create your weekly summary following [`../WEEKLY_TEMPLATE.md`](../WEEKLY_TEMPLATE.md)
2. Save to your folder: `weekly-summaries/YYYY-MM-DD-week-N.md`
3. Commit and push to GitHub by **Thursday 5:00 PM CST**
4. Notify Sysha that your summary is ready
